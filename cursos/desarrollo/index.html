Objetivo:

Con este curso adquirirás una sólida formación en el desarrollo de aplicaciones de escritorio bajo los lenguajes de C# y Java utilizando Visual Studio y NetBeans Respectivamente.

NOTA:

Las aplicacciones desarolladas sobre Java Utilizaran Postgresql como Base de Datos.

Las aplicacciones desarolladas sobre C# Utilizaran MS SQL Server como Base de Datos.

DIRIGIDO A:

Estudiantes o profesionales que quieran desarrollar aplicaciones de escritorio conectadas a Bases de datos.

Personas que desean reforzar sus conocimientos en dichos lenguajes.




REQUISITOS:

  • Computadora personal de escritorio y/o Laptop
  • Procesador i3 equivalente o superior
  • Memoria RAM 2GB o superior
  • Tener 40GB de espacio en Disco Duro
